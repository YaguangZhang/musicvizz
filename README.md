[![License: MIT License](https://img.shields.io/github/license/mashape/apistatus.svg)](https://opensource.org/licenses/MIT)

## Installation
1. Clone Repository
2. `cd` into directory
3. Run `python -m SimpleHTTPServer`
4. Go to `localhost:8000` in your browser
